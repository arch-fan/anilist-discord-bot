## Deploy

You already have a `docker-compose` file and a `Dockerfile` for deploying it easy. Also the `docker:deploy` script on the package-json.

Deploy command:

```bash
pnpm run docker:deploy
```

or using npm:

```bash
npm run docker:deploy
```
