## Deploy

You already have a `docker-compose` file and a Dockerfile for easy deployment. Also, the `docker:deploy` script is included in the package.json.

Deploy command:

```bash
pnpm run docker:deploy
```

or using npm:

```bash
npm run docker:deploy
```
